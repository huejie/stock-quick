# HEARTBEAT.md - 每日任务检查清单

## 📨 积压消息检查（最高优先级）

**触发条件：**
- 收到消息：`执行积压消息检查`
- 每30分钟自动检查一次（通过定时任务）

**执行步骤（严格按顺序）：**

1. **执行检查脚本：**
   ```bash
   node /root/.openclaw/scripts/backlog-checker-main.js
   ```

2. **读取检查结果：**
   ```bash
   cat /root/.openclaw/workspace/backlog-output.txt 2>/dev/null || echo "没有积压消息"
   ```

3. **如果第2步有输出内容：**
   - 直接将输出内容作为回复发送给用户
   - 不要附加任何其他文字

4. **如果第2步输出"没有积压消息"：**
   - 回复：`HEARTBEAT_OK`

**重要规则：**
- 这是最优先的检查项，收到此消息立即执行
- 发现积压消息立即发送，不要等待
- 记录推送状态，避免重复发送
