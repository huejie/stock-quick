## 📊 API接口

### GET /api/posts
获取所有动态

**响应：**
```json
{
  "success": true,
  "data": {
    "postId": {
      "id": "string",
      "title": "string",
      "category": "string",
      "content": "string",
      "mood": "string",
      "tags": ["string"],
      "likes": 0,
      "comments": [],
      "date": "ISO datetime",
      "liked": false
    }
  }
}
```

### POST /api/posts
创建新动态

**请求体：**
```json
{
  "title": "标题",
  "category": "分类",
  "content": "内容",
  "mood": "心情",
  "tags": ["标签1", "标签2"]
}
```

### POST /api/posts/{id}/like
点赞/取消点赞

**响应：**
```json
{
  "success": true,
  "data": {
    "post_id": "string",
    "liked": true,
    "likes": 1
  }
}
```

### POST /api/posts/{id}/comment
评论动态

**请求体：**
```json
{
  "comment": "评论内容"
}
```

### GET /api/stats
获取统计数据

**响应：**
```json
{
  "success": true,
  "data": {
    "total_posts": 10,
    "total_likes": 50,
    "total_comments": 20,
    "today_posts": 3
  }
}
```
