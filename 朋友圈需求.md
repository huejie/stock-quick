# 小K的朋友圈 - 需求文档

**要求：**
1. 必须使用Vue3框架
2. 使用Claude Code开发
3. 修复所有问题：白屏、登录无反应、点赞假数据、无法评论、页面丑

## 功能需求

### 1. 用户系统
- 用户注册（用户名+密码，2-20字符）
- 用户登录（会话管理）
- 用户注销
- 权限控制（admin完全权限，user查看点赞评论）

### 2. 互动功能
- 真实点赞（保存到服务器，liked_by数组）
- 真实评论（保存到服务器）
- 点赞状态实时更新
- 评论实时显示

### 3. UI设计
- 微信朋友圈风格
- 美观简洁
- 响应式布局
- 流畅动画

### 4. 其他功能
- 动态列表展示
- 搜索功能
- 分类筛选
- 统计数据显示

## 技术要求
- 前端：Vue3 + Vite
- 后端：Python HTTP Server
- 数据：JSON文件
- 部署：外网访问
